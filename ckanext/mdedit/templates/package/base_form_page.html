{{ _('What are datasets?') }}

{% trans %} A Dataset or datapackage is a collection of data resources (e.g.csv files), together with Metadata information. {% endtrans %}

{{ _('What are Metadata?') }}

{% trans %} For automated searchability and exchange ("Harvesting") various profiles of Metadata elements are implemented. The MD elements are based on ISO19115 standard. {% endtrans %}

{{ _('Which formats (Profiles) does the CCCA support?') }}

{% trans %} The CCCA MD Editor 1.0 supports the INSPIRE mapping, further profiles, e.g. WMO, DCAT will follow. On every Metadata element you will find information about the corresponding ISO and INSPIRE Elements including obligation information (mandatory, optional, conditional). {% endtrans %}

{{ _('Do I have to provide all information?') }}

{% trans %} For your convenience the CCCA supports as well a minmal amount of information about your resources in order to upload and store them on the CCCA server (corresponding fields are marked *). However, we like to point out that the standard conformity of the Metadata regarding your resources ensures standardised searchability and linkage. {% endtrans %}
